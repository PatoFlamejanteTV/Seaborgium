# Seaborgium
Web browser written in VB.NET with suppost to HTML/CSS/JS + IronPython + VB.NET. Written mainly using `Microsoft.Web.WebView2`.

## Building
Double-click (Open) `VBNETBrowserTest.vbproj` **>** `Build` **>** `Build solution`
